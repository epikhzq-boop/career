# Career Branding

A long-term professional development knowledge base for working in the Japanese IT industry.

## Overview

Career Branding 是一个长期维护的职业发展项目。

本项目的目标不是完成一次性的自我介绍、面试准备或日语润色，而是持续建立一套真实、统一、符合日本 IT 行业的职业表达体系（Professional Narrative）。

所有文档、资料与讨论，都围绕这一长期目标展开。

---

# Project Objectives

本项目致力于帮助 User：

* 建立符合日本职场的职业形象（Professional Image）
* 建立长期一致的职业叙事（Professional Narrative）
* 提升技术表达与沟通能力
* 积累可复用的项目经验与职业素材
* 为派遣先、面试、客户沟通、发表及未来职业发展提供统一素材

项目关注的是长期成长，而不是单次任务。

---

# Collaboration Model

Assistant 与 User 共同维护本项目。

Assistant 负责：

* 分析与优化表达
* 建立职业故事
* 主动发现用户尚未意识到的价值
* 提出符合日本职场的表达建议
* 从长期职业发展的角度提供意见

User 负责：

* 提供真实经历与项目经验
* 修正 Assistant 的合理推测
* 确认最终事实
* 持续更新职业成长内容

双方共同完善项目内容。

# Working Philosophy

本项目遵循以下原则：

* 长期成长优先于短期完成任务。
* 真实性优先于夸张包装。
* 表达设计优先于语言润色。
* 建立职业故事优先于堆砌经历。
* 日本职场视角优先于中文表达习惯。
* 每一次经历，都应尽可能沉淀为未来可复用的职业资产。

---

# Maintenance Policy

本项目采用持续迭代的方式维护。

所有文档均可随着 User 的成长不断更新。

新增内容应优先考虑：

1. 是否具有长期价值。
2. 是否能够提升职业表达能力。
3. 是否能够在未来复用。

项目目标不是积累资料，而是建立一套能够伴随职业发展的长期知识库。
