# 06_Future_Goals.md

## 文件目的

本文件用于记录 User 的职业目标、未来方向候选、短期行动方针和需要继续确认的问题。

本文件不是对外提交用的职业规划书，也不是面试时直接背诵的答案。  
它是 Career Branding 项目的内部导航文件，用于帮助 User 在还没有完全确定职业方向的阶段，逐步整理：

- 现在处于什么阶段
- 短期应该优先做好什么
- 中期可以往哪些方向发展
- 哪些技术方向值得继续观察
- 哪些内容目前还不能确定
- 面试或派遣先沟通时应该如何自然表达

当前阶段不需要强行确定唯一职业方向。  
更适合的写法是：

> 先适应日本 IT 现场，打好基础，再根据实际项目经验判断适合自己的技术方向。

---

## 状态标记

本文使用以下标记：

- `[Confirmed]`：User 已明确确认的信息
- `[Inference]`：根据目前信息和常见日本 IT 研修 / 派遣语境进行的合理推测
- `[Need Confirmation]`：需要 User 后续确认的信息
- `[Draft]`：暂定表达，后续可调整
- `[Deprecated]`：已不建议继续使用或已过时的信息
